# 第三节“柔软心”是实证的根本（P110-P116）

“柔软心”是由“思维根本状态”的“大悲心”演变而来的，带有“大悲心””的所有特征。当我们进入“思维衍生状态”后，“大悲心”的这些特征受“执着心力”作用而出现了不同程度的变化，比如：在深度方面没有“大悲心”深，在作用对象方面没有“大悲心”的作用对象普遍；此外，“大悲心”的“细、悬”也变成了现在这种心理状态的“粗、浮”。

我们需要对现在这种心理状态进行调整，使之逐渐形成“柔软心"。“柔软心”是由我们现在这种状态的“刚强心”向大悲心发展过程中过渡状态的心理特征。柔软心具有向心力，是能够引导我们证入高级思维状态甚至思维根本状态的根本力量，我们实证的方法虽然很多，但这些方法归根结底都是为了诱导和培养柔软心。

柔软心的培养分为“平时培养”和“实证诱导”两种方式。我们往往能够重视在实证中使用各种方法诱导和培养柔软心，但却容易忽视在平时生活中培养柔软心。这是根本错误的，因为实证是在潜意识主导下进行的，我们只有在平时生活中养成培养柔软心的习惯，这样才能形成稳定的、连贯的潜意识，才能真正在潜意识主导下进行实证。

有的人虽然也能重视在平时生活中培养柔软心，但由于长期生活在“执着心”主导的烦乱状态，因此已经形成牢固的不利于实证的生活习惯，这就必然导致我们往往无法及时发现问题，更无法真正解决问题。我们需要对这些问题进行深入分析，这样才能全面、深入地认识这些问题，才能具有针对性的解决这些问题。这些问题主要来自于以下几个方面：

1. 身体方面
身体是和现在这种思维状态对应的物质，因此只要我们仍然很在意身体，很善于使用身体，那么我们就是在培养执着心，而不是在培养柔软心。

2. 人我方面
人我方面指的是我们和别人的关系。在执着心力的作用下，我们总是执着于对立分别念中的第一个念头“我”，这就必然导致我们总是以自我为中心考虑问题，从而形成人我的对立。要知道这种人我的对立从根本上说是由于我们执着于对立分别念中的某个念头（也就是“我”这个念头）而导致的，是“执着心力”的一种表现，因此只要我们还有人我的对立分别观念，那么我们就是在培养执着心，而不是在培养柔软心。另外要注意，我们这里的“人”指的是各种动物，并不特指人类，因为它们和我们的区别仅仅在于所执着（选择）的物质体有所不同罢了，二者并没有本质区别。如果我们仍然把它们当做动物，那么我们就是在培养执着心。

3. 物我方面
物我方面指的是我们和物质的关系。物质是“思维根本状态（根本能量）”在“执着心力”作用下的波动状态、波动形式，因此从根本上说物质和我们是一样的，物质就是我们自己。如果我们仍然把物质当做物质，那么我们就是在培养执着心，而不是培养柔软心。

4. 思维方面
思维方面主要指“选择性思维”的高频率选择活动。“选择性思维”建立在“执着心”基础上，是形成我们现在这种思维状态的基础，是我们无法摆脱现在这种低级思维状态的根本原因，因此只要我们还在使用“选择性思维”，就是在培养执着心，而不是在培养柔软心。

通过以上分析，我们可以看出，对柔软心破坏最大的是执着心，是“我执”，因此在生活中我们要有针对性地解决这个问题，这样才能有效培养柔软心。

1. 身体方面

身体有很多状态，在现在这种思维状态，我们只能使用和现在这种思维状态对应的身体状态，反过来，只要我们还在使用身体的这种状态，那么我们就是在使用现在这种思维状态，就是在使用执着心，而不是在培养柔软心，因此对于身体在柔软心方面的影响，主要是使用还是不使用现在这种物质状态的问题。只要我们不使用现在这种身体状态，那么我们就是在培养柔软心。怎样才能不使用现在这种身体状态呢？我们主要从以下两个方面采取措施：

（1）从思维方面采取措施

在平时生活中，我们都能自如地使用身体，表面看来我们和身体是无缝对接的，但事实并不是这样，我们的每一个动作，都是一连串思维活动的结果，举例来说：我们伸出筷子夹菜，首先我们在心中起念想吃这个菜，然后又决定去取这个菜，这时候才开始指挥身体取这个菜，紧接着身体产生运动。

我们可以看出在身体运动之前还有几个念头，但由于这种活动太熟悉，我们能够非常熟练地完成整个过程，因此我们往往无法觉察到身体运动前的这几个念头，无法觉知到是我们在指挥身体运动，无法觉知这个“指挥与被指挥”的关系。

假若这个过程像做一道非常复杂的数学题那样困难，那么我们就能清清楚楚觉知整个过程中的每一个细节。事实正是这样，在我们从高级思维状态退下来的过程中，刚开始是没有身体觉受的，是体验不到身体的。只有当散乱心真正形成后，身体觉受才开始恢复，我们才能体验到身体的感觉、体验到身体的存在，但这时候我们并不能够像在平时生活中那样自如地使用身体，恰恰相反，这时候我们必须首先决定使用身体，然后还必须起念指挥身体运动，这样身体才能进行运作。在身体开始运动后，我们会发现，我们对身体的使用并不是非常自如的，需要经历一个“从生疏到熟练”的过程，也就是各种肢体动作刚开始是极其缓慢的，需要经历一段时间才能逐渐加速并恢复到平时生活状态。这个动作缓慢的阶段，就是我们和身体的结合过程。在这个阶段我们是可以随时放弃、完全放弃身体的运动的。

由此可见，我们要养成缓慢使用身体，使身体处于缓慢运动的习惯。只要身体能够处于极其缓慢的运动状态，我们将能够发现“指挥身体运动”的整个过程。只要能够发现这个过程，我们就能斩断这个过程，从而放弃对身体的使用。

为此，我们必须培养我们的觉知力，必须清楚这个过程中的每一个细节，必须熟练掌握这种“指挥与被指挥”关系，这样才能随时解除这种“指挥与被指挥”关系，才能随时放弃使用这个身体。只要我们能够自如地放弃使用身体，那么我们就是在放弃使用执着心，就是在培养柔软心。

（2）从微观物质方面采取措施

从医学的角度讲，身体的宏观运动过程是神经传导的过程，而从思维本体学的角度 讲，宏观物质运动过程是微观物质运动的表现形式，因为宏观物质是由微观物质生成 的，微观物质状态决定宏观物质状态，因此只要我们能够掌握微观物质运动规律，就能掌握使用身体的问题，就能随时放弃对身体的使用。

与身体密切关联的微观物质状态是气机状态，我们只要能够掌握气机的运动规律，就能掌握在使用身体方面的主动权。我们通过“观手”和“观呼吸”掌握气机运动规律，这样即使在运动中，只要我们能够停留在气机状态，那么我们就不再继续受身体宏观运动状态的困扰，比如：这时候即使蚊虫叮咬，我们也决不会有丝毫感觉，虽然此时我们仍然处于运动状态。在这种情况下，我们是处于稳定的、深沉的柔软心状态。

以上两种方法都能使我们自主、自如地使用身体，而不是相反地被身体束缚。只有这样我们才能既使用身体，同时又处于稳定的、深沉的柔软心状态，而不是像现在这样总是被动地、执着地使用身体，总是在使用执着心、培养执着心。

在生活中，当我们还没有真正做到上面两种方法所能达到的程度时，我们可以因时因地灵活使用一些减轻“执着心”的方法，比如：

当我们在照镜子的时候，我们要告诫自己这是在使用“执着心”。

当我们感受到蚊虫叮咬和疾病的痛苦时，我们要告诫自己这是在使用“执着心”。

当我们穿上漂亮的衣服而沾沾自喜时，我们要告诫自己这是在使用“执着心”。

当我们在品尝诱人的美食时，我们要告诫自己这是在使用“执着心"。

当我们在参加体育比赛等竞技性活动时，我们要告诫自己这是在使用“执着心”。

......

总之，在使用身体的时候，我们要随时警惕自己用“执着心”使用身体，而要相反地安 住于“柔软心”使用身体；在不需要使用身体的时候就要完全放弃身体觉受，只有这样才能科学使用身体，经常安住于柔软心状态。

2. 人我方面

人我方面最重要的是要从理论上了解我们每个人都有“思维根本状态”，而“思维根本状态”是无私无我的，是没有人我对立的。

在平时的生活中，我们要随时破除以自我为中心的思维习惯，比如：我们要经常告诫自己，实证的目的是为了他人，而不是为自己。

在人我方面尤其重要的是要平等对待一切动物，这里的平等对待指的是一定要明白动物也是人。在实证获得根本突破后，我们将发现一切动物的确和我们一样，平等无二，没有差别，都是鲜活的生命，都是人类。尽管我们现在还证不到这一点，但我们一定要这样观想。只有这样我们才能用柔软心看待一切动物，才不至于在看到动物时觉得自己比他们高级，才不至于在不知不觉中培养执着心，远离柔软心。

总之，在人我方面我们的所作所为，要以“思维根本状态”为标准，要与“思维根本状态”相应。只有这样才能回归“思维根本状态”，否则我们将常住低级思维状态，无法获得生命的自由和自主。

3. 物我方面

物我方面最根本的就是要在理论上明白我们和外物都诞生于同一个根本，都是由“根本能量”生成的，因此我们是一体的，这样我们就能在生活中视万物为自身，没有物我的对立。等我们的实证获得根本突破后，我们将真正体验到心物是一体的，一切物质和物质世界的确就是我们自己。只有到了这时候，我们才能真正破除物我的对立，才能形成稳定的、深沉的柔软心，不过这时候的柔软心已经不能再称之为柔软心了，而应该称之为“大悲心”。

4. 思维方面

在生活中，有时候我们必须使用“选择性思维”处理一些重要事情，而有时候并没有重要事情需要处理，这时候我们就要放弃使用“选择性思维”，因此在思维方面我们要根据具体情况灵活机动地使用“选择性思维”，及时破除“选择性思维”所带来的不利影响。具体操作方面分为“使用”与“不使用”选择性思维：

（1）不使用“选性思择维”：生活中有的时候不需要使用“选性思择维”，这时候我们通过“观心”的方法放弃使用“选性思择维”。

（2）使用“选性思择维”：生活在现在这种状态，有时候我们不得不使用“选性择思维”，这时候我们要注意以下两个方面：

A 降使低用频率

要降低“选择性思维”使用的频率，防止“选择性思维”处于高频选择活动中。这需要我们及时判断哪些情况需要使用“选择性思维”，哪些情况不需要使用“选择性思维”，在不需要使用“选择性思维”的时候要及时停止使用“选择性思维”，因为这时候是“选择性思维”最容易形成高频活动的时候。

B专注地使用“选择性思维” 有时候我们不得不使用“选择性思维”，这时候我们要正确认识“选择性思维”，不能一棍子打死。“选择性思维”对实证的有利之处在于这种思维功能容易形成专注，有利于培养专注力，因此当“选择性思维”确定观察对象后，我们就 要高度专注于观察对象，致力于培养专注力。

为了更好地培养柔软心，破除刚强心，我们要对自己的行为，尤其是习惯性行为进行 全面、深入地分析，每时每刻都要清楚自己当下的行为是“刚强心”还是“柔软心”。由于我们已经暗示自己要破除“刚强心”，因此只要我们能够发现 “刚强心”的活动迹象，那么潜意识就能自动排除“刚强心”，自然安住于“柔软心”。

在对自我的检查中，我们要注意防止执着于某一念头，尤其要注意防止住于“我执”，防止执着于“我”这个念头。另一方面我们要经常培养柔软心，比如：经常观想一些能够引起柔软心的场景。只要和柔软心能够培养起来，我们就能通过对比，更加轻松地发现“刚强心”的活动迹象，把“刚强心”消灭在萌芽中。

【鉴别】不要把“柔软心”当做了“懦弱”、“和柔弱”

“柔软心”是一种慈悲心，具有慈悲心的人是非常坚强的，因此不要把“柔软心”当做了懦弱、柔弱。

在实证中，只有与大悲心相应，才能证入“思维根本状态”，因此我们必须培养柔软心。

另一方面，在平时生活中，我们已经养成了各种不利于实证的不良习惯，久而久之这些习惯形成了不利于实证的不良潜意识，然而实证必须在潜意识主导下进行，这就必然导致潜意识无法正确运作，无法引导我们顺利完成实证，因此实证的首要任务就是必须改造潜意识，必须破除不良生活习惯、破除各种习性。

俗话说“江山易改本性难移”，战胜自己远比战胜敌人要困难得多；再加上刚开始实证时，我们必须面对、必须解决身体方面的各种不适和精神方面的昏沉散乱等，这并不是容易做到的。我们必须具有将军意志、大丈夫气概，才能破除这些不良习性和身心方面的各种问题。

由此可见，我们的实证必须同时具有慈悲心与将军意志，一个人要同时拥有这两个特质，是很不容易的。现代化的温暖之乡，已经磨去了人们的将军豪气。对于大多数人而言，培养慈悲心并不困难，但要培养将军意志，却不是一件容易事。但为了使实证能够快速获得突破，我们要想尽各种方法锻炼自己，培养坚忍不拔的意志力。只要我们的柔软心和意志力能够同时达到很高程度，实证将很容易突破。
